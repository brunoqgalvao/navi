# Ship It - Deploy to Navi Cloud

Deploy your app to production with a single command.

## Usage

When the user says "ship it" or wants to deploy:

1. Build the project for production
2. Deploy to Navi Cloud
3. Return the live URL (*.usenavi.app)

## Supported Frameworks

- Next.js
- Vite/React
- SvelteKit
- Static sites

## Example

> "ship it"

Your app will be deployed and available at `your-app.usenavi.app`
