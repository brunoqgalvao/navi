# Link to Global Skill

This template uses the global `vibe-coder` skill.

To use: The skill is already available globally at `~/.claude/skills/vibe-coder/SKILL.md`

The vibe coder pipeline includes:
1. **Spec Interrogator Agent** - Asks clarifying questions, generates SPEC.md
2. **Implementation Planner Agent** - Researches and creates IMPLEMENTATION_PLAN.md
3. **Code Implementer Agent** - Executes the plan step by step
