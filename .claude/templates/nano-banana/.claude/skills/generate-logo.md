# Link to Global Skill

This template uses the global `generate-logo` skill.

Location: `~/.claude/skills/generate-logo/SKILL.md`

## Quick Reference

```bash
# Simple logo
nano-banana generate "minimalist logo for [NAME], modern, clean lines" --pro -o logo.png

# App icon
nano-banana generate "app icon for [APP], gradient colors, geometric" --pro -o icon.png

# Wordmark
nano-banana generate "wordmark logo for [NAME], futuristic font, white background" --pro -o wordmark.png
```

Always use `--pro` for logos - quality matters for branding!
