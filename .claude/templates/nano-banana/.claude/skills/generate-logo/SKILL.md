# Logo Generator

Generate professional logos using AI image generation.

## Usage

When the user wants a logo:

1. Gather requirements (style, colors, concept)
2. Generate logo using Nano Banana Pro
3. Save in multiple formats (PNG, SVG if possible)

## Example Prompts

- "Create a logo for my coffee tracking app"
- "Design a minimalist logo for TechStartup"
- "Generate an app icon for my fitness app"

## Tips

- Use simple, bold designs that work at small sizes
- Consider both light and dark backgrounds
- Generate multiple variations for the user to choose from
